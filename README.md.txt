# Physics_1 Student Bot

This is a Streamlit-based chatbot for students studying Physics using the **Physics_1.pdf** textbook.  

The bot can:
- Answer questions
- Summarize sections
- Find topics by page number
- Solve equations and plot graphs
- Generate quizzes
